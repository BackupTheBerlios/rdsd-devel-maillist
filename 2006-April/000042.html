Well, if you've got a radio card with RDS decoder, you should be able to 
decode at least the basic RDS information. Usually, you only have to pay for 
special services transmitted via ODA groups.  But it is well possible that 
some stations don't send normal TMC groups but only ODA. You'll have to find 
out. By the way, specifying a good interface for ODA is still an open issue 
for rdsd/librds. Suggestions are welcome.
