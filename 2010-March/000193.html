> 
> Also the USB stick uses the si470x module. Anyone tried to use an
> si470x chipset with rdsd?

I doubt it. Prerequisite is that the kernel driver implements a read()
function for /dev/radioX. AFAIK this was only done for bttv and saa7134.
Simply do a cat /dev/radio0 and you'll know. If you get weird characters
on your terminal, it's OK. If you receive an error message, the kernel
driver needs fixing.
