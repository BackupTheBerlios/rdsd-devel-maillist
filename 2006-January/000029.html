The radio has three interface elements for the user: A
knob for setting the volume of the integrated
headphone amplifier (the headphone cable is also the
antenna), a button for automatic tuning to the next
station, and a second button used to store/recall a
favourite radio station.

The radio runs from either three dry batteries (4.5V)
or three NiMH batteries (3.6V).

You can connect it via serial port to the PC. And yes,
you can set the frequency from the PC, which is what
my kernel module does. There's also limited access to
the field strength and the stereo flag. The radio
sends raw demodulated RDS data via the serial port,
and the kernel module synchronizes on it and provides
the interface to Hans' rdsd software.

The radio chip itself is a TEA5768.

I wished it had a USB interface, this is maybe what we
add next time :-)
This project was meant to be built up by kids in our
local ham radio club, so we wanted as much 'big'
components as possible, the radio chip and a few SMDs
being the only chicken feed left to the experienced
guys.

I'm not totally up to date, but I believe that more
than 60 radios have been built up to now!
