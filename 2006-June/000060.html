Hello people,
im an italian developper.

I successfully installed my KNC-ONE Rds card on a Salckware 10.0 linux.
Those are the steps :
* Compiled kernel 2.6.17.1 with video4linux as Modules
* Downloaded last tree of video4linux drivers frow cvs on web
* Compiled it
* Make / Make install
* modprobe saa7134
* modprobe saa6588
* mknod /dev/video0 c 81 0
* mknod /dev/radio0 c 81 64
I used radio to set the freq and then the sample/test program that i found
at rdsd.berlios.de

Good luck to all
